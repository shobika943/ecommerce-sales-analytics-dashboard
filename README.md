# Ecommerce Sales Analytics Dashboard

## Project Overview

The **Ecommerce Sales Analytics Dashboard** is a data analysis project designed to analyze ecommerce sales data using **SQL and Python**.  
It generates meaningful insights and visualizations such as sales by city and product performance, and exports final analytical reports.

This project demonstrates skills in:
- SQL database design and querying
- Python data analysis
- Data visualization
- Report generation

---

## 🛠️ Technologies Used

- Python
- MySQL
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ⚙️ Requirements

- Python 3.8 or above
- MySQL Server
- MySQL Workbench (optional)
- Required Python libraries:
  - pandas
  - numpy
  - matplotlib
  - seaborn
  - mysql-connector-python

---

## 🗄️ Database Setup

1. Open **MySQL Workbench**
2. Run the SQL files in the following order:
   - `SQL/database.sql`
   - `SQL/insert_data.sql`
   - `SQL/analytics_queries.sql`
3. Ensure the MySQL server is running

---

## 📦 Install Python Dependencies

Run the following command in Command Prompt or Terminal:


pip install pandas numpy matplotlib seaborn mysql-connector-python


---

## ▶️ How to Run the Project

1. Open Command Prompt
2. Navigate to the project directory:


cd D:\Ecommerce_Analytics_Project


3. Navigate to the Python folder:


cd Python


4. Run the analysis script:


python ecommerce_analysis.py


---

## 📊 Output Generated

- Sales charts are saved in the `Charts/` folder
- Final analyzed data is saved in:
  - `Reports/final_sales_data.csv`
  - `Reports/final_sales_report.xlsx`
